SASS - Default Variables

$button-color: #D71C0D;
$button-color: #005E95 !default; // doesn't override

.button {
	padding: 1em;
	$background: $button-color;
}

SASS - Partials and Imports

// framework/_defaults.scss

$button-color: #005E95 !default;
$body-font: Helvetica, Arial, sans-serif !default;
$header-font: $body-font !default;
// _my-config.scss

$button-color: #005E95;
$body-font: "Lucida Grande","Segoe UI", Arial, sans-serif;
$header-font: "Avenir LT W01 35 Light", Arial, Helvetica, sans-serif;
// main.scss
@import "framework/defaults";
@import "my-config";
@import "framework";

body {
	font-size: 100%;
	font-family: $body-font;
}

h1, h2, h3, h4, h5, h6 {
	font-family: $header-font;
}

.button {
	padding: 1em;
	$background: $button-color;
}

Compass VS Bourbon

Compass

  • Big, requires extra installs
  • Complex typographic system
  • Comes with its own plugin system
  • Powerful tools that extend SASS

Bourbon

  • Lightweight, just copy .scss files
  • Just plain SASS
  • Lots of good common components
  • Simpler typography tools
